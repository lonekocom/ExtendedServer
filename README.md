# ExtendedServer

## English

**ExtendedServer** is a fork of **AdvancedServer** that combines the original source code with a collection of custom plugins and additional features.

The repository includes both the **AdvancedServer** codebase and plugins developed specifically for **ExtendedServer**, providing an extended and unified server platform.

---

## Русский

**ExtendedServer** — это форк **AdvancedServer**, объединяющий оригинальный исходный код с набором собственных плагинов и дополнительными возможностями.

Репозиторий включает как кодовую базу **AdvancedServer**, так и плагины, разработанные специально для **ExtendedServer**, предоставляя единую расширенную платформу для сервера.
